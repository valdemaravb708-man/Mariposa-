# Mariposa-
Dashboard de BI Logístico feito com Python (Dash/Plotly) e Pandas. Possui interface moderna em dark mode, sidebar retrátil com CSS nativo e filtros dinâmicos acoplados. Monitora 4 KPIs operacionais cruciais e 7 gráficos interativos de performance (receita, lucro e frotas), além de tabela analítica com formatação condicional de status.
